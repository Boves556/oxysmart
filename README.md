To run the code:
```
chmod +x setup_and_run.sh
```
then run:
```
./setup_and_run.sh
```

after it loads, run:
```
chmod +x second_script.sh
```
then run:
```
./second_script.sh
```

after it loads, run 'exit' then run:

```
chmod +x third_script.sh
```

then run:
```
./third_script.sh
```
